⌨️ VS Code Configuration and Extensions

  • 🔎 Open text files from the Terminal
  • 🩹 Fix formatting errors in your code

  • 1. Open VS Code
  • 2. Open the Command Palette (command + shift + p)
  • 3. Shell Command: Install 'code' command in PATH command
  • 4. Open the Extensions tab (command + shift + x)

  • 👇 Install the following extensions

  • 🧵 ESLint 💅 Prettier 🔭 GitLens 💞 Live Share 📈 GraphQL

⌨️ VS Code Configuration and Extensions


            npx install-peerdeps --global eslint-config-wesbos

  • 1. Open the Command Palette (command + shift + p)

              "workbench.colorTheme": "Default Light+",
